PROJECT ACCESS

Доступ к проектам платформы

Project Access — это индивидуальный уровень допуска к отдельным инвестиционным проектам и закрытым раундам платформы NoctAI. Решение о предоставлении доступа принимается вручную и основывается на совокупном участии инвестора.

Публичные проекты и их текущие статусы представлены в разделе Projects.

Перейти к Projects →

Условия рассмотрения

Запрос на Project Access рассматривается при совокупном участии от $10,000 в проектах платформы.

  • учитывается общий объём инвестиций, а не один транш
  • доступ не предоставляется автоматически
  • платформа оставляет за собой право отказа без раскрытия причин

Что предоставляет Project Access

  • доступ к закрытым и будущим проектам
  • ранний доступ к отдельным инвестиционным раундам
  • приватные обновления и материалы по проектам

Запрос на Project Access

Для рассмотрения запроса укажите контактный email и Transaction Hash (TxID), подтверждающий ваше участие в проектах платформы.

Все запросы рассматриваются вручную. В случае одобрения доступ предоставляется по указанному email.

Project Access не является гарантией участия в закрытых проектах. Все инвестиционные решения принимаются индивидуально.
