PROJECT ACCESS

Доступ к проектам платформы

Project Access — это расширенный уровень доступа к отдельным проектам и инвестиционным раундам платформы NoctAI. Доступ предоставляется по результатам индивидуального рассмотрения.

Условия рассмотрения

Рассмотрение Project Access возможно для инвесторов с совокупным участием от $10,000 в проектах платформы.

  • учитывается совокупный объём инвестиций
  • доступ не предоставляется автоматически
  • платформа сохраняет право отказа без объяснения причин

Что даёт Project Access

  • доступ к закрытым и будущим проектам
  • ранний доступ к инвестиционным раундам
  • приватные обновления по проектам

Запрос на Project Access

Для рассмотрения запроса укажите email для связи и Transaction Hash (TxID) инвестиции, подтверждающей участие.

Запросы рассматриваются вручную. В случае одобрения доступ предоставляется по email.

Project Access не является гарантией участия в закрытых проектах. Все решения принимаются командой NoctAI индивидуально.
